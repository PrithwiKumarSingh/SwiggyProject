# 🍔 Swiggy Clone — Food Ordering App (React)

🚀 A modern Swiggy-inspired food ordering web application built using React, powered by real-time APIs and designed with a clean, responsive UI.

## ✨ Features

- 🍽️ Real-time Restaurant Data using live Swiggy API  
- 🔍 Search functionality to find restaurants instantly  
- 🧭 Dynamic Routing with React Router  
- ⚡ Fast and responsive UI using Tailwind CSS   
- 🧠 Component-based architecture  
- 🔄 Shimmer UI / loading states for better user experience  

---

## 🛠️ Tech Stack

- Frontend: React.js  
- Styling: Tailwind CSS  
- Routing: React Router DOM  
- Language: JavaScript (ES6+)  
- API: Swiggy Live API

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/PrithwiKumarSingh/SwiggyProject.git
cd SwiggyProject

```
### Install dependencies && run 
```bash
npm install // dependencies install 
npm run dev  // run the project

```
